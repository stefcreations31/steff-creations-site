📡 Connexion au site (obligatoire)

Pour publier vos modifications sur le site, vous devez configurer l'acces GitHub.

📋 Comment obtenir un token:

  1. Allez sur github.com/settings/tokens/new
  2. Note: tapez "Stef Admin"
  3. Expiration: choisissez "90 days"
  4. ⚠️ COCHEZ LA CASE "repo" (tres important!)
  5. Cliquez le bouton vert "Generate token"
  6. Copiez le code qui commence par ghp_
